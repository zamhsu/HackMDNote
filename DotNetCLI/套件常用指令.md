# 套件常用指令

dotnet CLI指令
到專案目錄下

列出已使用的套件清單
```
dotnet list package
```

安裝/更新
```
dotnet add package <PACKAGE_NAME> [options]
```
安裝/更新至指定版本
```
dotnet add package <PACKAGE_NAME> -v|--version <VERSION>
```

移除
```
dotnet remove package <PACKAGE_NAME>
```
